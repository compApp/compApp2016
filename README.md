## RUNNING

##### Clone & Install

* ```shell
  git clone git@github.com:compApp/compApp2016.git
  ```


* ```shell
  cd compApp
  ```


* ```shell
  npm install
  ```

  ​

##### Android

* ```shell
  react-native run-android
  ```

  ​

##### iOS

* ```shell
  react-native run-iOS
  ```
