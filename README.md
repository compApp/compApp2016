## RUNNING

##### Clone & Install

1. ```shell
   git clone git@github.com:compApp/compApp2016.git
   ```

2. ```shell
   cd compApp
   ```

3. ```shell
   npm install
   ```

##### Android

* ```shell
  react-native run-android
  ```

##### iOS

* ```shell
  react-native run-iOS
  ```
